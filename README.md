###FastAPI Churn

**Задача сервиса** - предсказывать отток клиентов на основе их поведения и характеристик

**Что сделано**
- Подключение искусственного датасета
- Заполнение пропущенных значений
- Разбивка данных на train и test 
- Обучение моделей логистистической регрессии и случайного леса
- Предсказание на основе внешних данных
- Сохранение моделей и информации о них в .joblib и .json соответственно
